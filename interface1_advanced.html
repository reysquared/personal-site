This is the first variation of the JS-PY interface (advanced edition). It is constructed to fit a specific script on the server, and has a separate input field for each argument.

Associated files:

  • pyinterface1_advanced.js
  • cgi-bin/wrapper1_advanced.py
  • style.css (global stylesheet)

A simpler version of this interface concept is available here. It only supports the deciphering operation.



Caesar Cipher Solver

First choose an option (encoding or decoding.) Then fill out the relevant fields. (If you are deciphering the text, the field for rotation value has no effect.) Then hit "Run" and the script's output will appear in the bottom textbox.
