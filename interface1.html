This is the first variation of the JS-PY interface (standard edition). It is constructed to fit a specific script on the server, and has a separate input field for each argument.

Associated files:

  • pyinterface1.js
  • cgi-bin/wrapper1.py
  • style.css (global stylesheet)

A more advanced version of this interface concept is available here. It utilizes a drop-down menu to select which function from the .py file should be run.



Caesar Cipher Solver

Put the text to be solved in the input text box and hit "Run". The script's best guess at what the solution should be will appear in the output box.
